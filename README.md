# 🚢 Exploratory Data Analysis (EDA) on Titanic Dataset

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to uncover patterns and insights related to passenger survival. It focuses on understanding data distributions, relationships, and key influencing factors using visualization techniques.

---

## 📊 Dataset

* **Dataset:** Titanic Dataset
* **Description:** Contains passenger details such as age, gender, class, fare, and survival status

---

## 🎯 Objectives

* Analyze and understand dataset structure
* Handle missing and inconsistent data
* Perform univariate and bivariate analysis
* Identify key features influencing survival
* Visualize insights using graphs

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 🔍 Steps Performed

### 1. Data Loading

* Loaded dataset using Pandas
* Displayed initial rows and structure

### 2. Data Cleaning

* Handled missing values (Age, Cabin, Embarked)
* Removed unnecessary columns
* Converted categorical variables

### 3. Univariate Analysis

* Explored individual features using histograms and count plots
* Analyzed distributions of age, gender, and class

### 4. Bivariate Analysis

* Compared survival with gender, class, and age
* Used bar plots, box plots, and scatter plots

### 5. Correlation Analysis

* Generated heatmap for numerical feature relationships

---

## 📈 Key Insights

* Female passengers had significantly higher survival rates
* First-class passengers were more likely to survive
* Younger passengers showed slightly better survival chances
* Higher ticket fares were associated with higher survival

---

## ✅ Conclusion

The analysis highlights how factors like gender, passenger class, and fare influenced survival outcomes. These insights provide a strong foundation for building predictive models.
